Condição (if)

 A instruçõo 'if' é usada em situações em que um certo bit de código só é necessário para ser executado se determinadas condições forem atendidas. A sintaxe é a seguinte: 

                                            
                                                
if (condição){

                                                
 código;

                                                
}

Considere o seguinte exemplo:

                                            
                                                
var my_num = 5;

                                                
if(my_num < 6){

                                                
 document.write('Meu número é menor que 6 ');

                                                
}

                                                
 //Output: Meu número é menor que 6.

 No código acima, na verificaçõo condicional da instruçõo 'if', tentamos verificar se a variável my_num é menor que o número 6. Como my_num é menor que 6, ou seja, a verificaçõo condicional retorna true, o código dentro do parêntese da instruçõo if é executado.

Loop ( switch – case )

 Se houver várias verificações condicionais a serem realizadas, as instruções 'if … else if' podem ficar muito longas e tediosas. Uma abordagem melhor é usar a instruçõo ' switchcase '. Exemplo: 

                                            
                                                
switch ( parâmetro de condiçõo ){

                                                
 case n1:

                                                
  bloco de código 1;

                                                
  break;

                                                
 case n2:

                                                
  bloco de código 2;

                                                
  break;

                                                
 case n3:

                                                
  bloco de código 3;

                                                
  break;

                                                
 case n4:

                                                
  bloco de código 4;
    
                                                
  break;

                                                
 defaut:

                                                
  bloco de código padrão;

                                                
}

Considere o seguinte exemplo:

                                                                    
                                                
var select_color = 'Blue';

                                                
switch (select_color){

                                                
 case 'Red':

                                                
  document.write('Você escolheu Vermelho');

                                                
  break;

                                                
 case 'Blue':

                                                
  document.write('Você escolheu Azul');

                                                
  break;

                                                
 case 'Grenn':
                                                                   
  document.write('Você escolheu Verde');

                                                
  break;

                                                
 case 'Black':

                                                
  document.write('Você escolheu Preto');

                                                
  break;

                                                
 default

                                                
  document.write('Você precisa escolher uma cor');

                                                
  break;

                                                
}

                                                
// Output: Você escolheu Azul.

 Aqui a variável select_color é enviada para a instruçõo ' switch case' como o parâmetro condicional a ser verificado. O valor dessa variável é verificado sequencialmente para encontrar correspodentes. Aqui, um correspondente é encontrado no caso 'Blue'. Portanto, o código dentro desse bloco é executado. A instrução ' break' é usada para sair de um caso de switch depois que o correspondente é encontrado e o código acima é executado. Na sua ausência, o programa irá para os casos subsequentes e continuará procurando correspondentes.

 Se nenhum correspondente for encontrado, o código dentro do caso ' default' será executado.
