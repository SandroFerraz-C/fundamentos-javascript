Objetos (Introdução)

 Um objeto é uma coleção de dados e/ou funcionalidades relacionadas (que geralmente consistem em diversas variáveis e funções —— que são chamadas de propriedades e métodos quando estão dentro de objetos). Exemplo: 

                                            
                                                
var carro = {

                                                
marca: "Ford", Modelo: "Mustang",

                                                
tipo: "Coupe", tamanhoMotor: 5.2
                               
                                                
}

Fundamentos JavaScript

Propriedades e métodos de objetos

 Propriedades do objeto:
Os pârametros do objeto (variáveis) podem ser acessados das seguinte maneira: 

                             
                                
var carro = {

                                
marca: "Ford", Modelo: "Mustang",

                                
tipo: "Coupe", tamanhoMotor: 5.2

                                
}

        
                              
var x = carro.marca;

                              
var x = carro['marca'];

Métodos de objeto:
Os métodos de objeto podem ser aessados da seguinte maneira: 

                           
                            
NomeObjeto.nomeMetodo();

Para acessar os dois parâmetros e os métodos, o operador ponto ('.') é muito útil.

Fundamentos JavaScript

Criando objetos

 Depois de ter seu construtor de objetos, o comando 'new' pode ser usado para criar novos objetos: 

                           
                            
var carro = {

                            
marca: "Ford", Modelo: "Mustang",

                            
tipo: "Coupe", tamanhoMotor: 5.2

                            
}

        
                            
var carro_2 = new carro ("Chevrolet", "Camaro",  6.2);

                            
document.write(carro_2.modelo);

        
                            
//Output: Camaro

                          
                            
document.write(carro_2.tamanhoMotor);

        
                            
//Output: 6.2

Métodos de objeto:
Os métodos de objeto podem ser aessados da seguinte maneira: 

                         
                          NomeObjeto.nomeMetodo();

 Para acessar os dois parâmetros e os métodos, o operador ponto ('.') é muito útil.
