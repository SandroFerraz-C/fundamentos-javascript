introdução

 Uma função é um pedaço de código projetado para executar uma tarefa especifica ou ser reutilizado repetidamente. As funções ajudam a reduzir o comprimento do código e a melhrar o desempenho da aplicação. é possível chamar funções de qualquer parte do código para executar uma tarefa específica.

chamando uma função

 Uma função pode ser chamada de qualquer parte do programa para iniciar sua execução. Para chamar uma função, especifique o nome da função seguido por um conjunto de parênteses (). Exemplo: 

                                                                           
                                                
function impressao() {

                                                
document.write("Bom dia")

                                                
}

                                                
impressao();

                                                
//Output: Bom dia!

 Aqui a função 'impressao' é chamada mencionando seu nome e, em seguida, por um conjunto de parênteses ().

Caixa de alerta

 A função 'alert' pode ser usada para criar uma caixa de mensagem para notificar o usuário sobre um evento. Ele usa uma string ou uma varíavel de string como parâmetro, que é exibida como uma mensagem dentro da caixa de alerta. Exemplo: 

                                                                       
                                                
alert ("Este é um alerta!");

parâmetro de função

 Os valores podem ser enviados para outra função como atributos. Estes podem ser operados pela função. Exemplo: 

                                                            
                                                
function adicionar(a, b) {

                                                
document.write(a+b)

                                                
}

                                                
adicionar(2, 6);

                                                
//Output: 8

 No exemplo acima, a função 'adicionar' é chamada e 2 valores são enviados a ela. Esses valores são adicionados e impressos pela função. Uma fnção pode receber qualquer número de parâmetros.

Retornando um valor de uma função

 Uma função pode retornar uma valor para onde foi chamado usando a instrução 'return'. Exemplo: 

                                                                        
                                                
function adicionar(a, b) {

                                                
return a + b; }

                                                
}

                                                
var x = adicionar (1, 3);

                                                
document.write(x)
                              
                                                
//Output: 4

 No código acima, os valores 1 e 3 são enviados para a função 'adicionar'. Essa função adiciona esses 2 vaores e usa a instrução 'return' de volta ao local em que foi chamada. Aqui, o valor retornado é atribuído à variável x.
