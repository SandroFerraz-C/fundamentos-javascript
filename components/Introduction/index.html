História

 Em 1993, o desenvolvedor Marc Andreessen ajudou o Centro Nacional de Aplicativos de Supercomputação da Universidade de Illinois a lançar o Mosaic: Um navegador gráfico e amigável ao usuário. A internet estava evoluindo de uma área de entusiastas em texto para o universo multimí dia para os principais usuários de computadores.

 Posteriormente, Andreessen fundou a Mosaic Communications Corporation e publicou um navegador comercial chamado Mosaic Navegator em 1994 (O nome foi alterado pela Netscape apóis uma ação judicial.) O Netscape Navigator foi um sucesso e o Netscape estabeleceu novos recordes.

 A resposta da Microsoft, ( Internet Explore ), apareceu ameaçadora no horizonte. Andreessen sabia que o Navigator teria que empurrar novos recursos para manter o dominio; os navegadores da Web precisavam ir além da exibição de documentos estáticos e executar um softwareverdadeiramente interativo.

 Em 1995, a Netsape intermediou um acordo com outro concorrente da Microsoft, a Sun Microsystems, que estava lançando uma importante linguagem de programação, o java. O Java empresta a sintaxe do C ⁄ C++, mas o bytecode Java complilado é portatil − pode ser executado como esta em diferentes sistemas operacionais.

 A natureza restrita de Java significava que ainda havia a necessidade de um 'script ', de uma linguagem para aprimorar a página da Web; ⁄ Animação de menus suspensos, validação de entradas de formulários etc. Como a Sun posicionava o Java como a opção proficional para programas embarcados complexos, essa linguagem complementar foi vista como uma pequena solução acessí vel para designers e autorea de páginas. Fundamentalente, ele poderia ser escrito diretamente dentro dos documentos HTLM e seria interpretado pelo prõprio navegador Navigator.

 Foi nesse ponto no inicio em 1995 que Brendan Eich foi contratato pela Netsacape para projetar essa linguagem em um prazo extremo. Os requisitos fornecidos a Eich e suas decisões em cumpri-los acabariam tendo um efeito desproporcional monumental na natureza Web.

Fundamentos JavaScript

Introdução

 JavaScript, geralmente abreviado como JS, é uma linguagem de programação que esta em conformidade com a especificação ECMAScript.
Javascript é de alto ní vel, geralmente compilado na hora certa e com vários paradigmas. Possui sintaxe entre colchetes, digitação dinâmica, orientada a objetos com base em protótipo e funções de primeira classe.

 Juntamente com HTLM e CSS, o JavaScript é uma das principais tecnologias da Web. O JavaScript ativa páginas da Web interativas e é uma parte essencial dos apllicativos da Web. A maioria dos sites o utiliza para o comportamneto da página do lado do cliente, e todos os principais navegadores da Web possuem um mecanismo Javascript dedicado a eecuta-lo.

 Como uma linguagem de múltiplos paradigmas, o JavaScript suporta estilos de programação orientados a eventos, funcionais e imperativos.
Possui interfaces de programação de aplicativos ( APIs) para trabalhar com textos, datas, expressões regulares, struturas de dados padrões e o DOM (Document Object Model). No etntanto, a própria linguagem não inclui nenhuma entrada / saida (E /S), como rede, armazenamento ou recursos gr´ficos, pois o ambiente host (geralmente um navegador da Web) fornece essas APIs).

 Os mecanismos JavaScript foram originalmente usados apenas em navegadores da Web, mas agora estão incorporados em alguns servidores, geralmente via Node.Js. Eles também são incorporados a uma variedade de aplicativos criados com estruturas como Electron e Cordova.

 Embora exista semelhança entre JavaScript e Java, incluindo nome da linguagem, sintaxe e respectivas bibliotecas padrão, os dois idiomas são distintos e diferem bastante no disign.

Fundamentos JavaScript

Aplicações

JavaScript tem um grande número de aplicações. Alguns deles são:

  1. Desenvolvimento Web.

     JavaScript é uma linguagem de script de cliente usada para criar páginas da Web. é uma linguagem independente desenvolvida pela Netscape. é usado quando uma página da Web deve ser dinâmica e adiciona efeitos especiais em páginas, como: rollover, rollout e muitos tipos de gráficos. é usado principalmente por todos os sites para fins de validação.

  2. Aplicativos da Web

     Com a tecnologia, navegadores e computadores pessoias melhoram na medida em que eram necessário uma linguagem para criar aplicativos Web robustos. O JavaScript usa interfaces de programação de aplicativos (APIs) que fornecem poders extras ao código.

  3. Apresentações

     O JavaScript também oferece a facilidade de criar apresentações como um site. O JavaScript fornece a bibliotecas: RevealJS e BespokeJS para criar um Deck de slides baseados na Web. O Reavel.js cria alguns dos decks mais bonitos e interativos usando HTLM.

  4. Aplicativos de servidor

     O JS é criado no tempo de execução JavaScript do Chrome para criar aplicativos de rede escaláveis e rápidos. Ele usa aplicativos controlados por eventos, leves e eficientes, que devem ser distribuidos pelos sistemas com a ajuda de um navegador.

  5. Servidores da Web

     Usando o JS, um servidor da Web pode ser criado. As vantagens do JS é que ele é orientado a objetos e não esperaria pela resposta da chamada anterior. Ele passa para a próxima chamada e aproveita os eventos para receber notificações quando uma resposta é recebida para a chamada anterior. Os servidores criados no JS são muito rápidos e não usam buffer e transferem pedaços de dados. Além disso, ele é de threaded (JavaScript é single threaded) único com loop de eventos. Apenas uma coisa pode acontecer por vez, em uma única thread principal, e tudo é bloqueado até que a operação seja concluída.

  6. Games

     JavaScript também ajuda na criação de jogos para lazer. A combinação de JavaScript e HTLM também torna o JavaScript popular no desenvolvimento de jogos. Ele fornece a biblioteca Ease.JS, que fornece soluções simples para trabalhar com gráficos avançados.

  7. Arte

     Usar o HTLM5 em JavaScript desenhando gráficos em uma página da Web é teoricamente fácil. Todas as formas bidimensionais e tridimensionais podem ser facilmente desenhadas em uma tela e isso abriu o navegador como um novo meio para todos os diferentes projetos de arte digital.

  8. Aplicação Smartwatch

     JavaScript, é a linguagem mais usada, isso porque esta sendo usado em todos os dispositivos e aplicativos possiveis. O uso do JavaScript fornece a biblioteca Pebble.JS, usada em aplicativos Smartwatch.
