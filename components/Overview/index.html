Criando e adicionando JS a uma Página da Web

 O código JavaScript pode ser adicionado a um documento HTLM usando a tag <script>. O código JS pode ser colocado dentro da tag <head> ou <body> 

                                            
                                                
<html>

                                                
 </head>
            
                                                
 <body>

                                                
 <script>
      
                                                
  document.write("Olá Mundo!");
 
                                                
 </script>

                                                
</body>

Resultado: Olá Mundo!

 No código acima, o código JS está incluído entre a tag <script>.
O método document.write() é usado para imprimir na tela.

Incluindo um arquivo JS Externo

 Como o código JS e HTLM5 pode ficar longo e complexo, podemos dividir adicionando o código JavaScript como um arquivo .js externo. Vamos adicionar um arquivo .js que comtém o seguinte código ao nosso html:
document.write("Olá arquivo JS externo!"); 

                                            
                                                
<html>

                                                
 <head>

                                                
 </head>

                                                
<body>

                                                
   <script= "my_javascript.js">
                        
                                                
    </script>

                                                
  </body>

Resultado: ("Olá arquivo JS externo!");

 Usamos o atributo 'src' para fazer referência a um arquivo javascrit externo, no caso do exemplo acima chamado de 'my_javascript.js'. Lembre-se de incluir o arquivo na mesma pasta que o arquivo html, para fornecer o caminho do arquivo relativo ao arquivo html.

Fazendo um comentário no código JavaScript

 As linhas podem ser comentadas dentro do código Javascript de duas maneiras: 

                                 
                                                                                 
                                                
 //Este é um comentário de linha única
                                               
                                                
 /*Este é um

                                                
comentário de

                                                
varias linhas */

 Os comentários podem ser usados para melhorar a legibilidade de seu código e fazer com que o navegador ignore o código não utilizado.
