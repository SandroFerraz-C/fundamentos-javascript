Criando variáveis no JavaScript

 As variáveis podem ser ensinadas como recipientes onde os valores podem ser armazenados. Valores variáveis, como o próprio nome sugere, podem mudar ao longo do programa. A palavra-chave 'var' é usada para declarar variáveis de seguinte maneira: 

                                            
                                                
var minha_variavel = 5;
           
                                                
document.write(minha_variavel);

Resultado: 5

 No código acima, usamos a palavra-chave 'var' para declarar uma variável chamada 'minha_variavel' e atribuá-la um valor de 5.

 As seguintes regras devem ser seguidas ao declarar uma variável:

  1. O nome da variável deve começar com uma letra, um sumblinhado( _ ) ou um cifrão ( $ ). Os caracteres de nome subsequentes podem ser letras, dígitos, sublinhados ou cifrões.

  2. O nome não deve conter espaços.

  3. O nome não pode começar com um número.

  4. O nome não pode incluir símbolos matemáticos ou qualquer cracter especial como: '+', '%', '!', '*' e etc.

  5. Certos nomes são reservdos do JS como palavras-chaves e não podem ser usados, como: 'avstract', 'arguments', ' true', 'wrile' e muitas outras.

Tipo de Dados

 Os tipos de dados referem-se aos tipos de valores que um programa pode armazenar e usar. JS suporta tipos de dados como números, strings, boolenos, arrays e etc.

Segue alguns exemplos:

                                    
                                        
var meu_num = 5;

                                        
var meu_float_num  = 5.5;

                                        
var minha_string = 'Olá';

                                        
var meu_boolean_1 = true;

                                        
var meu_boolean_2 = false;

 Em JS, existem números com pontos decimais ou sem, ao contrário dos tipos de números como inteiro, float, double e etc.

 Variavéis do tipo strings são declaradas em volta de aspas simples (') ou aspas duplas (").

Operadores de Strings

 As cadeias podem ser unidas usando o operador ( + ).

Segue exemplo:

                                        
                                            
var strings_1 = "Meu nome é "

                                            
var strings_2 = "Sandro Ferraz"

                                            
var strings_3 = strings_1 + strings_2;
                                
                                            
document.write(strings_3);

                                            
// Output: Meu nome é Sandro Ferraz;

 No código acima, strings_3 é criada adicionando a strings_1 com a strings_2.
